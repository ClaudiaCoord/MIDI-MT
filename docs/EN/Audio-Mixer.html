Managing audio sessions of running applications

Additionally, it is possible to manage audio sessions of running applications. Control can occur either from a MIDI keyboard or from the built-in sound control panel.

The audio session control panel has several themes. You can choose the panel design according to your taste and the theme of your desktop. Also, there is a custom theme, in which you can change all the colors as you wish.

The following panel themes are available:

  • Light,
  • Dark,
  • Metro,
  • Retro,
  • Modern,
  • Customizable

The placement of the panel on the screen has four fixed initial positions:

  • vertical - right,
  • vertical - left,
  • horizontally at the top,
  • horizontally below

You can move the panel in the plane of its location at your own discretion. In other words, panels in a horizontal position can be moved vertically, and panels in a vertical position can be moved horizontally. The panel allows you to adjust the volume and mute for each session. Adjustment is possible both from a MIDI keyboard and mouse, using the built-in audio panel.

To control audio sessions from a MIDI keyboard, there is no need to call up the panel. The panel is only needed to adjust the sound using the mouse, or to visualize the status of current audio sessions.

Settings

The settings are made in control editor. To add an application whose volume you want to control, you need to add a new control in the editor or select an existing one. Edit it. In the target field you must specify the mixer identifier, this is the number 254, or the identifier of the multimedia button controller, this is the number 253. In the long-target field, you must indicate the purpose control, what it will regulate. On the left, there are tips in the editor that will help you choose the right value.

After this, you must add the full path to the managed program to the list of applications. The editor itself will convert it into the required format during processing; this value will be the application identifier. You can add an application through the file selection dialog by pointing to the desired executable file on the disk, or through the loaded applications button. In the latter case, all running applications that have an open audio session will be added. The extra ones can be deleted.

The number of managed applications per control is unlimited. But we must remember that if several applications are loaded, specified to one control, then the sound adjustment will occur only for the last one.

It is especially important to use the built-in mixer to control sound in Windows 11, where the usual means of adjusting sound have been removed, and access to existing ones involves a long journey through the settings menu.
