MIDI-MT Documentation in English

ABOUT THE PORTABLE VERSION

The portable version is packaged with Enigma Virtual Box, which sometimes gives false positives to antivirus programs. First on the list, of course, is Microsoft Windows Defender.

You can always assess the level of danger yourself. Link to scan results on virustotal.com.

If you have any doubts about the “purity” of the portable version of the file, simply do not use it.

  • Assembly MIDIMT_x64_Portable - requires preliminary installation of MIDI drivers.
  • Assembly MIDIMT_x64_PortableI - will allow you to use the portable version without first installing MIDI drivers.

REQUIRED DEPENDENCIES

To successfully run MIDI-MT, the following components are required:

  • Virtual MIDI driver from Tobias Erichsen.
  • the latest version Visual C++ Redistributable from Microsoft.

You can read more about installing these components in the Dependencies section.
