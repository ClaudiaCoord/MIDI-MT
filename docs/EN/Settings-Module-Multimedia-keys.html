MIDI-MT - Multimedia keys control module

MIDI-MT Documentation in English

Managing “multimedia keys”

Allows you to assign standard multimedia commands to controls on MIDI Control Surface. To successfully use these features, the managed application must be in focus.

Multimedia keys control module.

This module has no settings, the list of available commands is given below, it is also available in the control editor.

MIDI key Windows command
SYS_Zoom ZOOM
SYS_Scrub LAUNCH MEDIA SELECT
SYS_Record PLAY
SYS_Rewind MEDIA PREV TRACK
SYS_Forward MEDIA NEXT TRACK
SYS_Play MEDIA PLAY PAUSE
SYS_Up VOLUME UP
SYS_Down VOLUME DOWN
SYS_Left VOLUME MUTE
SYS_Right PAUSE

Control Binding Settings

The multimedia keyboard group number is “253”.

  • Configuration file

It is also possible to edit the configuration file in any editor you are familiar with. The main section of settings for managing multimedia keys looks like this:

    ...
    },
